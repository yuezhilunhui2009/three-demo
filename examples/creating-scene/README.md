# creating-scene 创建场景

## 必备知识
* 术语：3D坐标系、3D图形、顶点、三角形图元、网格（模型）、材质、光源、摄像机、透视、视口、着色器；
* 3D坐标系：包含 3 个互相垂直的坐标轴的坐标系，webgl 3D 坐标系中横轴为 x 轴，纵轴为 y 轴，垂直于屏幕的轴为 z 轴（就像 CSS 的 z-index 的方向一样）；
* 3D图形：使用 3 个维度来表示计算机几何数据的图形，数据使用三维坐标系表示，最终以 2D 图像渲染在计算机屏幕上；
* 顶点：即 3D坐标系中的一个点，具有 (x, y, z) 坐标；
* 三角形图元：由 3 个顶点定义的图形，三角形是组成 3D 世界的最基本图形（也有用四边形的系统）；
* 网格（模型）：由多边形（通常是三角形图元）拼接而成，通常我们将其称知为“模型”，注意网格是没有外观属性的，之所以在建模软件中网格能被看到只是为了方便设计师对网格进行编辑；
* 材质：网格的外观属性统称材质，材质定义了网格表面受到光源照射时对各种颜色光的反射比例；
* 光源：定义了场景被照亮的模式，通常光源跟材质两者的相互作用使 3D模型有了外观；
* 摄像机：以用户视角查看 3D 场景的观察点；
* 透视：通俗的说就是远处的物体看着比较小，近处的物体看着比较大，模拟真实世界人眼的观察效果；
* 视口：最终 3D 场景会渲染到 2D 的显示平面上，这个平面上显示区域的大小就是视口，如果将来可以将 3D 场景以 3D 方式渲染到真实世界中，可能定义会有不同（例如全息投影）；
* 着色器：定义了每个像素最终渲染结果的逻辑；

## 一个 THREE.js 3D 场景的基本要素
* 场景：THREE.Scene 的实例，用以容纳场景内的所有物体，包括模型、摄像机、光源；
* 模型（几何体 + 材质）；
* 光源；
* 摄像机；
* 渲染器；
